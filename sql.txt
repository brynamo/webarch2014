No it is not possible. This is MySQL instance and subsequently it follows ACID rules. This has to do with consistency and isolation. First the database goes from one consistent state to the next meaning that it is an all or nothing transaction, if there are errors nothing happens which means there would not be partial data entry. Secondly isolation prevents another user seeing what I am doing until the transaction is complete.
